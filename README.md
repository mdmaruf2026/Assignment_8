```markdown
# BookBorrow - Online Book Borrowing Platform

## Live URL

https://assignment-8-6cdn.vercel.app/

## Purpose
A seamless and modern web application designed to digitize the traditional library experience. Users can explore a vast collection of books, filter by categories, and borrow titles digitally.

## Key Features
- Browse and search books by title
- Filter books by category (Story, Tech, Science)
- User authentication with email/password and Google OAuth
- Book details with borrow functionality
- Private routes for authenticated users
- User profile with update functionality
- Responsive design for mobile, tablet, and desktop

## Tech Stack
- Next.js 16
- Tailwind CSS + DaisyUI
- BetterAuth
- MongoDB Atlas
- Swiper.js

## NPM Packages Used
- better-auth
- mongodb
- mongoose
- daisyui
- swiper

## Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account

### Installation
1. Clone the repo
2. Run `npm install`
3. Create `.env.local` with your credentials
4. Run `npm run dev`
```
